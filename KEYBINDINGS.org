#+TITLE: Doom Keybinds Cheat Sheet

* Global
- SPC o t :: Open vterm
- SPC o f :: Toggle Treemacs file tree
- SPC o a :: Org Agenda
- SPC o c :: Org Capture
- SPC o C :: Calendar
- SPC o o :: Open Obsidian vault (dired)

* My Menu (SPC m)
- m n :: Capture Note
- m t :: Capture Todo
- m j :: Journal (datetree)
- m a :: Agenda
- m o :: Vault in dired
- m T :: Terminal
- m F :: File tree
- m r :: Python REPL
